POS NETWORK PRINTER

Overview

POS NETWORK PRINTER is used to handle printing to ESC/POS network printers from Odoo POS Retail or Restaurant. This module replaces POS BOX or IOT BOX and you can use this on a local server or online server with POS offline mode. The module works with Odoo Enterprise or Community Edition.
Modules included : for Odoo v13 and newer, all theses modules are included in this module

  • POS RESTAURANT NETWORK PRINTER
  • POS NETWORK PRINTER ONLINE
  • POS RESTAURANT ONLINE

Configuration

Install the module from Applications. Set the configuration

  • Give access to Printer Connector for the user
  • Create Connector on Point of Sale / Configuration / Printer Connector
  • Create Printer on Point of Sale / Configuration / Network Printers
  • Set Printing mode (Offline for local server - Online for VPS, Cloud or Odoo SH)
  • Set Connector (JsPrintManager - Web Socket)
  • Set Ticket print mode (Web : default printing - Network : using network printer)
  • Set Bill print mode (Web : default printing - Network : using network printer) for POS Restaurant only
  • Set Orders Printers for POS Restaurant only
